# ESP32-C3 Project for ohmsprint by SN60 team.


Visualisation and data collection form uart protobuff messages.

## Setup

1. Install PlatformIO if needed:
   ```bash
   pip install platformio
   ```
2. Build and upload:
   ```bash
   pio run --target upload
   ```

## Usage

Check `Makefile` for usefull targets.
